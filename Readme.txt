=============================
PROJETO: GERENCIAMENTO DIGITAL
=============================

Autor: Mateus
Contexto: Programador iniciante, aprendendo HTML, CSS, JavaScript, Python e C++.
Objetivo: Criar um sistema de gerenciamento de funcionários com auxílio de IA, 
publicar no GitHub e evoluir como projeto futuro.

-----------------------------
O QUE O PROJETO RESOLVE
-----------------------------
- Controle de ponto digital (entrada e saída dos funcionários).
- Registro de faltas e cálculo de horas extras.
- Transparência para empresa e funcionário.
- Automatização de relatórios sem necessidade de controle manual.
- Interface diferenciada para gestores (mais elegante e profissional).

-----------------------------
LINGUAGENS USADAS
-----------------------------
1. HTML (HyperText Markup Language)
   - Responsável pela estrutura do site.
   - Tags usadas: <html>, <head>, <body>, <header>, <nav>, <div>, <span>, <button>, <table>, <tr>, <td>.
   - Cada tag define uma parte da página (ex: <header> é o cabeçalho, <table> cria tabelas).

2. CSS (Cascading Style Sheets)
   - Responsável pelo estilo visual.
   - Comandos usados: background-color, color, font-family, margin, padding, border-radius.
   - Define cores, fontes, espaçamento e aparência dos botões e tabelas.

3. JavaScript
   - Responsável pela lógica e interatividade.
   - Variáveis: let funcionarios = []; (array que guarda os funcionários).
   - Funções:
     - adicionarFuncionario(): adiciona novo funcionário à lista.
     - atualizarTabela(): atualiza a tabela com os dados atuais.
     - registrarEntrada(): registra hora de entrada do último funcionário.
     - registrarSaida(): registra hora de saída e simula horas extras.
   - Comandos usados:
     - document.getElementById(): acessa elementos HTML pelo ID.
     - innerText / innerHTML: altera conteúdo do site dinamicamente.
     - new Date().toLocaleTimeString(): pega a hora atual.

4. Python (futuro)
   - Será usado para lógica mais avançada (cálculo real de horas extras, integração com banco de dados).
   - Exemplo: scripts para gerar relatórios automáticos em PDF.

5. C++ (futuro)
   - Pode ser usado para integração com sistemas locais (ex: monitoramento de tempo em VS Code).
   - Exemplo: programa que coleta dados e envia para o sistema web.

-----------------------------
COMO FUNCIONA (LÓGICA)
-----------------------------
1. Usuário (gestor) abre o site.
2. Adiciona funcionários pelo formulário.
3. Funcionários aparecem na tabela com status inicial.
4. Botões "Entrada" e "Saída" registram horários.
5. Sistema calcula faltas e horas extras (simulação inicial).
6. Relatórios podem ser gerados e visualizados.
